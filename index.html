VDF research

Hello! This page keeps track of public material relevant to VDFs and groups of unknown order. The VDF Alliance is a collaborative effort to design and implement VDFs in software and in hardware.

To make changes to this page please submit a pull request to the Github repo or reach out to vdf@ethereum.org
